Mobile app iOS Android Branding

One Stop Club is a mobile app concept with an iOS and a Material Design version. The main idea behind the app is to serve as a hub for finding dance events from various sources - Resident Advisor, Facebook events, Eventbrite, etc - in one handy app.

Users can find and save events, add them to their phone calendar, as well as look up artists and promoters. This concept served as a stimulating exercise in using different mobile design guidelines.

About One Stop Club

Onestop club aims to be the “one stop” for finding all dance events in one convenient location without users having to install multiple apps or scouring various websites to find what is out there in terms of alternative/underground nightlife by collecting events from various sources such as Resident Advisor, Facebook events and Eventbrite.

The app is accessible through either iOS or Android devices with a dedicated version using the platforms’ respective design guidelines. Users can find and save events, add them to their phone calendar, as well as look up artists, clubs and promoters.

User Flow

Following the competitive analysis of event apps on the market to assess the scope of the project, I quickly designed an original user flow with only 6 screens to conceptualize this small project in a minimal amount of time. Throughout the multiple iterations the tasks and number of screens was significantly refined, resulting in the final user flow seen here.

Defined Features

  • Aggregated events from across the internet
  • Skippable account setup with basic search function
  • Personalized local feed
  • Search function by date and location
  • Shopping bag preview page
  • Follow favorite artists, clubs and promoters

Low Fidelity Wireframe Sketch

With the user flow in mind and after reviewing the respective design guidelines for each platform I began designing some wireframe sketches for both versions of the app.

Mid-Fidelity Wireframes

Using iOS and Material Design component kits from the Figma community I translated the sketches into digital wireframes. Typography, typeface (SF Pro text under 20 and SF Pro Display above 20 for iOS; Roboto for Material Design) and a majority of components (using Figma community kits) follow iOS and Material Design guidelines as much as possible. Custom components were created when necessary.

Colors

The primary color scheme was inspired by Sanzo Wada’s color combination number 046 (A Dictionary of Color Combinations, 2010, p. 48). With the app being aimed at nightlife events the predominant use of dark colors seemed the most logical with orange offering a striking contrast.

Usability Test

The final design stage involved the creation of 2 prototypes for each version of the app that were subsequently shared with 5 UI designers for evaluation. Testers were asked to walk through the app before commenting on design and flow. 5 people were tested in total (3 for iOs version and 2 for Android/Material Design). Based on the feedback received high fidelity wireframes were tweaked to finalize the design. The prototypes were updated before being submitted for a final round of review.

Conclusion

What went well?

Feedback on the app was very positive, particularly regarding color scheme and functionality. Keeping the scope of the project relatively small with the concept for an event app was also advantageous since the emphasis of this task was learning how to use the appropriate guidelines for each platform.

What were the challenges?

As an iOS user designing for Android was particularly difficult, having to take extra time to get familiar with the guidelines. Changes from low to high fidelity wireframes are far more noticeable compared to the iOS version. After this first experience I feel more confident using Material Design guidelines.

What could be improved?

More time for user testing would have been useful in order to make several iterations that improve on each other. Although testers liked the color scheme it might be difficult for colorblind users to navigate, further testing focusing on this aspect and an alternative color scheme would be useful avenues to explores.
