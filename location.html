Location Setting (Shpock)

Part of a major update of the mobile and web secondhand marketplace Shpock's location settings. This update was strategically designed to address critical user feedback and competitive benchmarks, focusing on the ease of listing and searching items based on preferred locations.

Designed user flows, low fidelity wireframes and mockups of several overhauls to onboarding, listings and searching. Designed from the ground up (both UX and UI)

responsibilities

UX Research, UX Design, UI Design

Project Planning

The update was driven by the insight that users required a more flexible and intuitive way to set their locations. This was evident from user reviews which highlighted issues such as difficulties in changing locations, problems with address recognition, and the necessity of sharing exact addresses.

Design Process

Our team undertook a holistic approach to reimagine the location settings. The process included:

  • User Flows Analysis: To ensure a seamless transition between different location setting options.
  • Low-Fidelity Wireframes: To outline the basic structure of the new location features.
  • High-Fidelity Mockups: To visualize the overhauled onboarding, listing, and searching experiences.

Key Features

  • Onboarding:Users now have the option to share their precise location or manually enter a ZIP code during the initial app setup.
  • Listing Flexibility: Sellers can set their item's location to an exact address, which is internally used to display the item within a radius, ensuring privacy.
  • Enhanced Search: The search function has been simplified with an easy location setting feature, allowing users to filter results by their preferred area.
  • Address Book: Users can now save multiple addresses, such as home and work, for quicker listing and searching.

Conclusion

The implementation of these features addressed the pain points identified in user reviews and competitor benchmarks. The new location settings empower users with better control over their privacy and improve the accuracy of listings and searches.

Onboarding has become more flexible, offering users the choice to share their location automatically or manually. For sellers, the update has made listing items more intuitive, with the ability to choose their exact address without disclosing it publicly. For buyers, the updated search function now includes a location filter that maintains the users' search radius based on a ZIP code.

The comprehensive redesign of Shpock's location settings in marks a pivotal improvement in user experience. By incorporating user feedback and addressing competitive gaps, the platform now offers a more personalized and user-friendly interface that streamlines the process of buying and selling on a secondhand marketplace.
